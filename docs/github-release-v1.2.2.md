# Sauki Pay WordPress Plugin v1.2.2

This release adds a local admin dashboard for standalone payment form submissions.

## Download

Download the plugin ZIP:

https://github.com/SAUKI-RESOURCE-LTD/SAUKIPAY-WORDPRESS-PLUGIN/releases/download/v1.2.2/saukipay-wordpress-plugin-v1.2.2.zip

## What's New

- Added Sauki Pay > Form Payments admin page.
- Added local WordPress database table `wp_saukipay_form_payments`.
- Saved standalone payment form submissions with payer name, email, phone, amount, currency, reference, environment, checkout URL, and status.
- Updated local form payment records after callback verification.
- Added filters for status, environment, currency, date range, and search.
- Added summary cards and manual verify action for form payments.

## Supported Payment Options

- WooCommerce checkout gateway.
- GiveWP donation gateway.
- Standalone Sauki Pay payment form using `[saukipay_payment_form]`.

## Installation

1. Download `saukipay-wordpress-plugin-v1.2.2.zip`.
2. Log in to WordPress admin.
3. Go to Plugins > Add New > Upload Plugin.
4. Upload the ZIP file.
5. Activate or replace Sauki Pay.
6. Open Sauki Pay > Form Payments after a standalone form payment is submitted.
